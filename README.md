# AI-UFD: public code and reproducibility materials

This repository accompanies the manuscript “Artificial Intelligence–Based Uroflowmetry Device: Prospective Clinical and Experimental Validation”.
 
AI-UFD is an investigational, portable, smartphone-enabled supplementary platform for decentralized uroflowmetry monitoring, potentially home-oriented assessment under stable acquisition conditions, and low-flow screening. 
 
## Data availability
 
The de-identified clinical analysis dataset and the peristaltic-pump bench validation dataset supporting the study are available through the restricted-access data mechanism described in the manuscript Data Availability statement.
 
Direct identifiers, raw patient videos, exact clinical encounter dates, and variables that could reasonably enable participant re-identification are not included in this repository.
 
## Code availability and protected components
 
The code and materials in this repository are manuscript-associated public resources. Some AI-UFD source code and model-related materials are not publicly released because they are subject to ongoing clinical-trial, approval, institutional, regulatory, and intellectual-property procedures.
 
Access to non-public components may be considered by the corresponding authors upon reasonable request, subject to institutional approval, applicable regulations, participant privacy protection, and intellectual-property restrictions.
 
## Clinical-use boundary
 
The materials in this repository are provided for research transparency and documentation. They are not intended for clinical deployment, diagnostic use, treatment recommendation, regulatory submission, or replacement of standard uroflowmetry or clinician interpretation.
 
Any clinical, home-based, or real-world use of AI-UFD requires further validation, appropriate acquisition-quality control, institutional approval, regulatory review where applicable, and clinician-in-the-loop interpretation.
 
## Citation
 
If you use this repository, please cite the associated manuscript and this repository. Citation metadata are provided in `CITATION.cff`.
 
## Security and privacy
 
Please do not submit issues, pull requests, discussions, screenshots, videos, or files containing personal health information, identifiable participant data, patient videos, private server details, credentials, tokens, certificates, or other sensitive information.
 
Potential privacy or security concerns should be reported privately according to `SECURITY.md`, not through public GitHub issues.
