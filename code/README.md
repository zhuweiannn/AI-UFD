# R statistical analysis and figure-generation scripts
 
This directory contains manuscript-associated R scripts and supporting materials for statistical analysis, validation summaries, and figure-generation workflows for the AI-UFD study.
 
The materials are provided to improve transparency of the analytical workflow. They should be interpreted together with the manuscript, tables, figures, Supplementary Information, Data Availability statement, and Code Availability statement.
 
This directory does not contain identifiable participant data, raw patient videos, exact clinical encounter dates, training data, model weights, Android native inference code, compute-service algorithms, or proprietary AI-UFD components.
 
The manuscript reports statistical analyses performed in R version 4.2.2. Users should verify package versions, input-file paths, and data-access permissions according to their own research environment and the manuscript Data Availability statement.
