# Non-core AI-UFD engineering materials
 
This directory contains public, non-core AI-UFD engineering materials released to improve transparency around the software-facing structure and reproducibility boundary of the project.
 
The materials may include interface-level code, user-interface workflow logic, example configuration files, API stubs, data-format descriptions, illustrative example input/output files, screenshots, and documentation.
 
This directory does not contain the complete AI-UFD analytical system. It excludes model weights, training datasets, patient videos, raw clinical datasets, Android native inference code, compute-service algorithms, protected reconstruction logic, private server endpoints, credentials, certificates, database configurations, or commercial/proprietary modules.
 
These materials are not intended for clinical deployment, diagnostic use, treatment recommendation, regulatory submission, or replacement of standard uroflowmetry or clinician interpretation.
